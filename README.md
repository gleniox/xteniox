# Xteniox

Xteniox is a sample **NuGet package** project showcasing how to build, test, and publish reusable libraries for the .NET ecosystem. It provides a foundation for creating third‑party components that can be integrated into other applications.

---

## 📦 Features
- Organized solution structure with **library** and **test** projects.
- Example **unit tests** using xUnit.
- Ready for **NuGet packaging** and distribution.
- GitHub Actions workflow for CI/CD automation.

---
